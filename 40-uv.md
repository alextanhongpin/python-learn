# Init a new package

```
uv init --package example-pkg
uv pip install -e example-pkg
```

## Create custom project

Example [here](https://github.com/alextanhongpin/python-uv-greet).

```
uv init --package .
```

Update the name to `greet` in `pyproject.toml`. The directory containing the source code should also be `src/greet`.

```
uv add 'git+https://github.com/alextanhongpin/python-uv-greet'
```

Running:
```python
from greet.greet import hello


def main():
    hello("john")


if __name__ == "__main__":
    main()

```
